## Where to buy R36s <a name="where-to-buy-r36s"></a>

The R36s is available from retailers such as Amazon.ie, TEMU, and others. Please ensure you conduct your own due diligence when purchasing (lots of R36S Clones out there, search web on how to identify clones)

**Potential Retailers**:

- Amazon
- AliExpress [R36s : **cannot guarantee authenticity, appears to be genuine**](https://www.aliexpress.com/item/1005007445996906.html)
- TEMU [R36s Clone : seller claimed original, but was a **clone**](https://share.temu.com/fbjzfku1UbB)
- Retro gaming specialty stores
- [Adverts.ie listings](https://www.adverts.ie/for-sale/q_r36s/)

**Verification Tips**:

1. Check product reviews
2. Compare photos with known non-clone units
3. Ask seller about panel type (thats the key to finding out)
4. If prices significantly below market rate then its probably too good to be true

**Clones vs Authentic**:

- Clones have different panel types
- May require different ArkOS images see flashing SD card
- Performance will probably be slower

### Useful Accessories <a name="useful-accessories"></a>

| Accessory | Purpose | Notes |
|-----------|---------|-------|
| **USB-C to USB-A Adapter** | Connect USB devices to R36S | Essential if USB port works |
| **USB Hub** | Connect multiple devices keyboard, WiFi dongle | Powered hub recommended, to run WiFi and Charge R36S |
| **USB Keyboard** | Terminal access on R36S | Any basic keyboard works USB C or with USB to USB C Converter |
| **WiFi Dongle** | Wireless connectivity | No built-in WiFi |
| **MicroSD Card Reader** | Flash/access SD card | USB 3.0 for faster transfers |
| **USB-C Cable** | Charging and data | Good quality recommended |
| **Carrying Case** | Protection for R36S | Optional but recommended |
| **Screen Protector** | Protect display | Cut to size |
